NIFTY SELLER LITE V1.2 — HEDGED STRIKE PLANNER

MAJOR ADDITIONS
- Added one post-decision protected strike planner.
- Added CE Sell candidate: OTM short CE plus farther-OTM long CE hedge.
- Added PE Sell candidate: OTM short PE plus farther-OTM long PE hedge.
- Added protected Iron Condor candidate with two short legs and two hedge legs.
- Added estimated credit points, wing width, maximum-risk points and breakeven levels.
- Added candidate-quality scoring from delta, bid/ask spread, OI, volume, distance from spot, support/resistance clearance and OI-wall context.
- Increased the local option-chain working window from ATM±5 to ATM±7 so protected hedge legs can be selected more reliably.
- Added planner statuses: READY, ALTERNATIVE, WATCH ONLY, BLOCKED, REFERENCE ONLY and UNAVAILABLE.

ONE-BRAIN INTEGRITY
- analysis/decision.py remains the only final strategy brain.
- analysis/trade_plan.py runs only after the final decision.
- The planner cannot alter CE/PE/Condor/WAIT scores or the final action.
- WAIT and reference-only sessions block execution planning.
- No broker order-placement code was added.

RELEASE HYGIENE
- Patch contains only changed/new files.
- No cache, pyc, runtime state, secrets or obsolete release files are included.
