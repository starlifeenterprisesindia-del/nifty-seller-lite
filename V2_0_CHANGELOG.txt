NIFTY SELLER LITE — V2.0 COMPACT EVIDENCE & INTEGRITY

MAIN UI
- Added a six-row All Features — Compact Evidence table at the top.
- Table order: Price Action; OI & Options Flow; EMA/MACD/RSI; Levels & Volume;
  Top-7 & FII/DII; VIX/Data/Event Risk.
- Directional rows normalize Bullish + Bearish + Neutral to 100.
- Evidence values are display evidence, not trade-profit probability.
- Final One-Brain Decision remains immediately below the table.
- Long decision evidence and cautions moved into a collapsed detail table.
- Planner, execution, position guardian, detailed core, detailed options and raw
  market data remain present inside compact expanders.

INTEGRITY
- Added analysis/evidence_matrix.py as a presentation-only aggregator.
- It does not feed analysis/decision.py and cannot become a second brain.
- Zero, negative or missing India VIX is now INVALID / UNAVAILABLE.
- Invalid VIX is labelled VIX DATA UNAVAILABLE, never BALANCED PREMIUM ENVIRONMENT.
- Live-session invalid VIX adds a bounded WAIT penalty and strategy caution.
- Snapshot JSON now converts nested datetime and time values to ISO-8601 strings.

HYGIENE
- Old V1.8 release documents are not carried in the clean V2.0 release tree.
- No order-placement or broker-write functionality was added.
- No duplicate Dhan API request or second strategy decision path was added.
