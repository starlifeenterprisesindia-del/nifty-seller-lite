INTEGRITY AND SAFETY
--------------------
- No extra option-chain API request was added. The existing chain response is reused.
- Analysis modules remain network-free.
- Closed/pre-market/reference snapshots are not saved into intraday option history.
- State contains no Dhan credentials, tokens, order data or account information.
- State retains only the current date and is capped at 180 snapshots.
- Quick identical snapshots are deduplicated.
- Distant historical samples cannot masquerade as 1m/3m/5m deltas.
- India VIX is dynamically resolved from the Dhan instrument master; configured fallback
  is IDX_I security ID 33.
